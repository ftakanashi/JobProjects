## 题目描述
给你一个会议时间安排的数组 intervals ，每个会议时间都会包括开始和结束的时间 intervals[i] = [starti, endi] ，返回 所需会议室的最小数量 。

示例 1：
```
输入：intervals = [[0,30],[5,10],[15,20]]
输出：2
```
示例 2：
```
输入：intervals = [[7,10],[2,4]]
输出：1
```

提示：
```
1 <= intervals.length <= 104
0 <= starti < endi <= 106
```

### 解法 差分数组
不解释……很明显用差分数组套路做的题。

当然注意因为给出的会议时间是离散的，所以不能真的用一个列表去维护时间的信息。
我这里用了哈希表来维护各个时间节点的差分值。
最后遍历排序过的哈希表中的各个时间点来求重叠最多的时候累计需要了多少会议室。
