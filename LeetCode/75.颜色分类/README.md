## 题目描述
给定一个包含红色、白色和蓝色，一共 n 个元素的数组，原地对它们进行排序，使得相同颜色的元素相邻，并按照红色、白色、蓝色顺序排列。

此题中，我们使用整数 0、 1 和 2 分别表示红色、白色和蓝色。
 

示例 1：
```
输入：nums = [2,0,2,1,1,0]
输出：[0,0,1,1,2,2]
```
示例 2：
```
输入：nums = [2,0,1]
输出：[0,1,2]
```
示例 3：
```
输入：nums = [0]
输出：[0]
```
示例 4：
```
输入：nums = [1]
输出：[1]
```

提示：
```
n == nums.length
1 <= n <= 300
nums[i] 为 0、1 或 2
```

进阶：
- 你可以不使用代码库中的排序函数来解决这道题吗？
- 你能想出一个仅使用常数空间的一趟扫描算法吗？

### 解法 双指针
说是双指针，其实是三指针吧。

首先l和r分别记录数组中0和2的边界位置，剩下一个i则从左到右扫描。

当i碰到0时，将其交换到l位置并l+=1。
当i碰到2时，将其交换到r位置并r-=1。
当i碰到1时，无视之继续让i往前走。

注意到这三个值虽然是对称的，但i是从左往右扫描的，所以实际上碰到0的时候，
i也应该往前走。
