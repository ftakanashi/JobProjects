## 题目描述
给你一个整数数组 nums，将 nums 中的的所有偶数元素移动到数组的前面，后跟所有奇数元素。

返回满足此条件的 任一数组 作为答案。

示例 1：
```
输入：nums = [3,1,2,4]
输出：[2,4,3,1]
解释：[4,2,3,1]、[2,4,1,3] 和 [4,2,1,3] 也会被视作正确答案。
```
示例 2：
```
输入：nums = [0]
输出：[0]
```

提示：
```
1 <= nums.length <= 5000
0 <= nums[i] <= 5000
```

### 解法 排序
题目不难。
按奇偶排序，偶数在前奇数在后，其实就可以是按`num & 1`排序。
所有偶数的结果都是0，奇数结果都是1，所以可以直接排出来。

再加上Python中的sort函数自带key参数，所以一步到位。
