## 题目描述
给定一个非空的字符串 s ，检查是否可以通过由它的一个子串重复多次构成。

示例 1:
```
输入: s = "abab"
输出: true
解释: 可由子串 "ab" 重复两次构成。
```
示例 2:
```
输入: s = "aba"
输出: false
```
示例 3:
```
输入: s = "abcabcabcabc"
输出: true
解释: 可由子串 "abc" 重复四次构成。 (或子串 "abcabc" 重复两次构成。)
```

提示：
```
1 <= s.length <= 104
s 由小写英文字母组成
```

### 解法 模拟
很简单的题目。
思路就是暴力。

当输入字符串是某个子字符串的重复时，单元子字符串最长只可能到原字符串长度的一半。
那么就依次遍历所有可能的单元串。

遍历过程中还可以直接跳过那些长度无法被源字符串长度整除的情况。
