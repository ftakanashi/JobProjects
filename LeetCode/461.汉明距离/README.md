## 题目描述
两个整数之间的汉明距离指的是这两个数字对应二进制位不同的位置的数目。

给出两个整数 x 和 y，计算它们之间的汉明距离。

注意：
0 ≤ x, y < 231.

示例:
```
输入: x = 1, y = 4

输出: 2

解释:
1   (0 0 0 1)
4   (0 1 0 0)
       ↑   ↑

上面的箭头指出了对应二进制位不同的位置。
```

### 解法2 快速统计异或结果中1的个数
位数不同，那不就是异或的定义么。

这样的话，其实这个问题可以转化为统计`x ^ y`的1的个数呗。
而这个问题可以参考`JZ15.二进制中1的个数`这个题目的解法。

核心是，`x & (x-1)`的结果，相当于把`x`最靠右的1改成0。
因此只要看改多少次后x成0了即可。
