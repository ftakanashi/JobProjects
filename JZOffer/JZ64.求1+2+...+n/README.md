## 题目描述

求 1+2+...+n ，要求不能使用乘除法、for、while、if、else、switch、case等关键字及条件判断语句（A?B:C）。

示例 1：
```
输入: n = 3
输出: 6
```
示例 2：
```
输入: n = 9
输出: 45
```

限制：
- 1 <= n <= 10000

### 解法 好多种全写一起了
这是个脑洞题，能想到的方法肯定越多越好。

#### 递归
既然不能用乘除和循环关键字，那么只能化循环为递归了

#### Python流氓解法
不解释了…sum(range)完事

#### 乘方+位运算
求和公式`n*(n-1)/2`中有障碍的是一个乘号一个除号。

乘号乘进去其实就是一个乘方，因此可以用乘方运算即`n ** 2`，而除号是除以2，自然可以位运算。

#### 快速乘
快速乘算法是用加法+位运算模拟乘法的一种算法。
