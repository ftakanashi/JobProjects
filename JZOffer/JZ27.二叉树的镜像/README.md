## 题目描述
请完成一个函数，输入一个二叉树，该函数输出它的镜像。

例如输入：
```
     4
   /   \
  2     7
 / \   / \
1   3 6   9
```
镜像输出：
```
     4
   /   \
  7     2
 / \   / \
9   6 3   1
```
 

示例 1：
```
输入：root = [4,2,7,1,3,6,9]
输出：[4,7,2,9,6,3,1]
```

### 解法1 递归（DFS)
这题首先有个共识，就是in-place地修改输入的树是比较方便的。

递归时显然，我将一个节点的左右子树递归地取镜像，然后互换两者位置就完成了本节点的镜像操作。

### 解法2 递推（BFS)
递归DFS改成利用队列的递推BFS，不解释
